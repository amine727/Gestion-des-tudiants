#  Gestion simple des étudiants (SQLite Android)

## 🎯 Objectif

Développer une application Android permettant de gérer des étudiants en utilisant une base de données locale **SQLite**.

---

## 🧩 Fonctionnalités

* ➕ Ajouter un étudiant (Nom, Prénom)
* 🔍 Rechercher un étudiant par ID
* ❌ Supprimer un étudiant
* 📂 Stockage local avec SQLite

---

## 🏗️ Architecture

Le projet est organisé en 3 couches :

* **classes** → modèle `Etudiant`
* **service** → logique CRUD (`EtudiantService`)
* **util** → base de données (`MySQLiteHelper`)

---

## ⚙️ Technologies utilisées

* Java
* Android SDK
* SQLite
* Android Studio

## 🚀 Installation

1. Cloner le projet :

```bash
git clone https://github.com/ton-repo/gestion-etudiants.git
```

2. Ouvrir avec Android Studio

3. Lancer l’application sur un émulateur ou un téléphone

---

## 🧪 Exemple d’utilisation

* Saisir un nom et prénom → cliquer sur **Ajouter**
* Entrer un ID → cliquer sur **Chercher**
* Cliquer sur **Supprimer** pour supprimer un étudiant
